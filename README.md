# Monaco - Tienda de Perfumes

Este proyecto es una aplicación móvil desarrollada en **Android Studio** utilizando **Kotlin**. La aplicación permite la gestión de productos de perfumería, incluyendo la visualización de artículos, su detalle, carrito de compras y un sistema básico de almacenamiento de datos.

## Descripción general

La aplicación fue creada con el objetivo de simular una tienda en línea de perfumes llamada **Monaco**, donde el usuario puede:

- Visualizar productos con su imagen, nombre y precio.
- Agregar o eliminar productos del carrito.
- Ver el total de la compra.
- Realizar un proceso de compra básico.
- Guardar datos localmente.

## Tecnologías utilizadas

- Android Studio (Kotlin)
- RecyclerView
- CardView
- Intents y Activities
- SharedPreferences (almacenamiento local)
- Diseño con ConstraintLayout
